Homebrew-cask to Install Simple Apps

May 31st, 2013

An interesting extension to homebrew can be found here on GitHub http://bit.ly/11tA8JQ

Its called homebrew-cask and will allow you to install software like Chrome, Alfred2, VLC Player etc. from the command-line in the usual homebrew fashion via

Basically homebrew-cask is based hombrew unilizing an extended repository with collective stored information where to download some software (i.e. VLC Player, Firefox) directly from the vendors-server, once downloaded as a .zip/.dmg file the payload gets installed will by shown as symlinke in a ~/Applications directory , yes thats the user domain aka NSHomeDirectory().

The final path for an application installed by hombrew-cask is ##/opt/homebrew-cask/Caskroom///.app

Its a pretty new project and of course you’ll might run into some issues but it’s not meant to be replacing Munki, Jamf CasperSuite or other solutions on the shelf.

  • Firefox v21
  • Calibre
  • GitHub.app
  • VLC

MacMini as Main Dev Machine

May 31st, 2013

Just to share how to setup a pretty fast and reliable developer machine with a Apple MacMini Server QuadCore i7 . The box will be based on a FusionDrive setup, no extra skills required appart folowing the iFixIt tutorial, turnaround time will be 45 minutes, you’ll need a torx and a 3rd party SSD replacement drive as extra parts for the MacMini. Thus far I’ve only tried this two times, everytime used an previous unbooted mini but see no problem to start such set up later ony any Thunderbolt MacMini.

  1. get a MacMini Server with i7 Processor
  2. turn upside down, open the cover
  3. follow instuctions to replace the hard drive for from the mini: http://bit.ly/18A62NA
  4. replace the drive with a decent SSD
  5. finish the instructions above but don’t boot yet
  6. connect the exctracted hard drive via external USB case to the mini
  7. boot from the external USB
  8. go thru OS X setup assistant as usual, create a account
  9. login to the account
  10. now you’ll see that diskutility tells you there is a problem with the drive
  11. diskutility offers to fix the problem for you

… et Voila!
You’ll get automagically build a FusionDrive by OS X Mountain Lion for free, without mumbling in the Terminal at all so we’re following the koan ‘less code is better code’ and on top you’ll save some code kittens for more important tasks – pretty cool isn’t it ?

To finish things up transfer some user data, install Xcode and your other dev tools required – you’ll have a decent fast machine in about two hours ready for any development task. best not forget to upgrade RAM to 16GB and you’re ready to run at top speed at a unbeatable price.

Final Specs:

  • MacMini 2.6 GHz Intel i7 QuadCore
  • 16 GB Ram
  • 1,5 TB Fusion drive
