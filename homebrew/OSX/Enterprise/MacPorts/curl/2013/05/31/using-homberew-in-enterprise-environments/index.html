Using Homberew on OSX in Enterprise Environments

May 31st, 2013

Ever tried to to install software via homebrew while you’re in enterprise environment or other networks with a web proxy present? Glad you have but without knowing in advance you’ll might run into a problem to try installing or updating via homebrew, reasion is pretty simple due to curl failing with its defaults to get along with the proxy.

Here is a short wrap to install/run homewbrew in such environment

Prerequisites:

read homebrew project page https://github.com/mxcl/homebrew/wiki/Installation

homebrew install instructions behind a proxy:

open Terminal.app

base requirements:

  • Xcode4 command line utilities installed
  • working ruby 1.8.7 or higher environment ( check OSX systems installed in Terminal: ruby -v)
